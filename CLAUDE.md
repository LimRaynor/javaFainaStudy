# 프로젝트 정보


## 학습자 공부  방식
- Java 기초를 **주제별 패키지**로 나누어 학습 중 (SWCAMP 과정)
- 각 Java 파일명은 `영어_한글` 형식으로 개념을 설명 (예: `Variable_변수.java`)
- 코드 안에 **주석으로 설명**을 달아가며 학습
- 각 패키지마다 `_요약정리.md` 파일로 내용을 정리

## 패키지 구조
```
src/javaFainal/
├── data/               # 데이터 기초 (변수, 상수, 자료형, 형변환, 출력, 문자열포맷)
├── operator/
│   ├── conditionalStatemen_조건문/  # 조건문, 연산자, 입력, 랜덤
│   └── Loop_반복문/                 # for, while, do-while, break, continue
├── ob/                 # 객체지향 (클래스, 생성자, static, 오버로딩)
└── Array/              # 배열 (배열, 배열초기화, ArrayList, NullPointerException)
```

## 작업 규칙
- 코드 오류 수정 시 `[수정]` 주석을 달아 원래 뭐가 틀렸는지 표시
- 요약정리 md 파일은 해당 패키지 폴더 안에 생성
- 새 패키지가 추가되면 같은 형식으로 요약정리 md 생성 요청 가능
- 수정사항이 생기면 요약정리 md도 함께 업데이트

## 응답 규칙
- **허위 정보 금지**: 실제 검증된 사례 기반으로만 답변. 추측/지어낸 내용 절대 금지.
- **핵심 코드 설명**: 코드 제공 시 핵심 부분을 추출하여 단계별로 쉽게 설명.
- **전체 흐름 설명**: 전체 로직 흐름과 구조(어떻게 연결되고 동작하는지)를 항상 설명.
- **서식 제한**: 이모지 사용 금지. 마크다운은 최소한으로(헤더, 코드블록, 볼드 정도만).
- **응답 언어**: 모든 응답은 반드시 **한국어**로 작성.
