# Java 기초 - 반복문 패키지 요약 정리

---

## 1. for문

- **반복 횟수가 정해져 있을 때** 주로 사용
- 구조 : `for (초기화; 조건; 증감)`

```java
for (int i = 0; i < 10; i++) {
    System.out.println(i);   // 0 ~ 9 출력
}
```

| 구성 요소 | 설명                          |
|-----------|-------------------------------|
| `int i=0` | 초기화 - 변수 선언            |
| `i < 10`  | 조건 - 반복 지속 조건         |
| `i++`     | 증감 - 한 블럭 실행 후 변화   |

---

## 2. while문

- **조건이 참인 동안** 계속 반복
- 반복 횟수가 불확실할 때 사용

```java
int i = 0;
while (i < 10) {
    System.out.println(i);   // 0 ~ 9 출력
    i++;
}
```

---

## 3. do-while문

- while문과 유사하지만 **최소 1번은 무조건 실행**
- 조건 검사를 **나중에** 수행

```java
int i = 0;
do {
    System.out.println(i);   // 0 ~ 9 출력
    i++;
} while (i < 10);
```

---

## 4. break

- 반복문을 **즉시 종료**
- 특정 조건에서 반복을 멈추고 싶을 때 사용

```java
for (int i = 0; i < 10; i++) {
    System.out.println(i);
    if (i == 6) {
        break;   // i가 6이면 반복문 종료 (0~6 출력)
    }
}
```

---

## 5. continue

- 현재 반복을 **건너뛰고** 다음 반복으로 진행
- 특정 조건일 때 해당 회차만 스킵

```java
for (int i = 0; i < 10; i++) {
    if (i == 6) {
        continue;   // i가 6일 때 건너뜀
    }
    System.out.println(i);   // 6을 제외하고 0~9 출력
}
```

---

## 반복문 비교 요약

| 반복문       | 특징                        | 사용 시기                   |
|--------------|-----------------------------|-----------------------------|
| `for`        | 초기화/조건/증감 한 줄 작성 | 반복 횟수가 정해져 있을 때  |
| `while`      | 조건만 검사                 | 반복 횟수가 불확실할 때     |
| `do-while`   | 최소 1번 실행 보장          | 일단 한 번 실행 후 판단할 때|
| `break`      | 반복 즉시 종료              | 특정 조건에서 멈출 때       |
| `continue`   | 현재 회차 건너뛰기          | 특정 조건만 제외할 때       |
